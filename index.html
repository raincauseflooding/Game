Explanation

This game has an infinite mathematical expected value. If we calculate the expected value \( E[X] \):

\[ E[X] = \sum_{k=1}^{\infty} P(\text{First Head is on Toss } k) \times \text{Reward}_k = \left(\frac{1}{2}\right)(1) + \left(\frac{1}{4}\right)(2) + \left(\frac{1}{8}\right)(4) + \dots = \frac{1}{2} + \frac{1}{2} + \frac{1}{2} + \dots = \infty \]
The Paradox: While mathematics claims the game is worth an infinite entry fee, almost no rational person would pay even $20 to play it once. This paradox was resolved by Daniel Bernoulli in 1738 by introducing Utility Theory: people evaluate risk based on marginal utility (the psychological value of money decreases as wealth increases) rather than pure expected monetary value.
